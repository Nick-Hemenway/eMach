class des_opt.DesignProblem(designer: des_opt.Designer, evaluator: des_opt.Evaluator, design_space: des_opt.DesignSpace, dh: des_opt.DataHandler)¶

Bases: object

Class to create, evaluate, and optimize designs

Attributes:
designer: Objects which convert free variables to a design. evaluator: Objects which evaluate the performance of different designs. design_space: Objects which characterizes the design space of the optimization. dh: Data handlers which enable saving optimization results and its resumption.
fitness(x: tuple) → tuple¶

Calculates the fitness or objectives of each design based on evaluation results.

This function creates, evaluates, and calculates the fitness of each design generated by the optimization algorithm. It also saves the results and handles invalid designs.

Args:
x: The list of free variables required to create a complete design
Returns:
objs: Returns the fitness of each design
Raises:
e: The errors encountered during design creation or evaluation apart from the InvalidDesign error
